# Hospital Readmission Analysis

## Project Overview

This project analyzes hospital readmission data using Microsoft Excel to identify trends and generate insights that support better healthcare decision-making.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Dashboard Design
* Data Visualization

## Analysis Performed

* Monthly hospital readmission trend analysis
* Patient age distribution analysis
* KPI dashboard creation
* Identification of readmission patterns

## Files Included

* Hospital_Readmission_Analytics_Dataset.xlsx
* Hospital_Readmission_Dashboard.png

## Objective

To demonstrate healthcare data analysis skills and build an interactive dashboard for decision-making insights.
